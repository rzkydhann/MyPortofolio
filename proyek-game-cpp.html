Game Shooter Sederhana C++

Deskripsi Proyek

Ini adalah game shooter sederhana yang berjalan di terminal/konsol, dibuat sepenuhnya menggunakan bahasa pemrograman C++. Proyek ini dibangun dengan menerapkan konsep-konsep utama dari Object-Oriented Programming (OOP).

Setiap entitas di dalam game, seperti `Player`, `Bullet`, dan `Enemy`, direpresentasikan sebagai *class* yang terpisah. Hal ini membuat kode lebih terstruktur, mudah dikelola, dan dapat dikembangkan lebih lanjut. Game ini mencakup logika dasar seperti pergerakan pemain, menembak, pergerakan musuh, deteksi tabrakan, dan sistem skor.
